# Public repository privacy

- This repository is public. Never publish real usernames, computer names, domains, employer or customer names, private drive layouts, webhook URLs, tokens, credentials, or diagnostic archives.
- Use obvious neutral placeholders such as `username`, `CONTOSO`, `WORKSTATION01`, and `C:\Path\To\...`; do not preserve a sensitive value merely because it appeared in a screenshot or log supplied for diagnosis.
- Screenshots and copied console transcripts require the same sanitization as source code. Prefer generated fixtures or demo-mode captures over live-machine output.
- Keep explanations outside copy/paste command blocks so every command block remains syntactically valid.
